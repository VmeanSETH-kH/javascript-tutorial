Sets (集合)

A JS Set is a collection of unique values.
Each value can only occur once in a Set.

Essential Set Methods

Method Description
new Set() Create a new Set
add() Adds a new element to the Set
delete() Removes an element from a Set
has() Returns true if a value exists in the Set
forEach() Invokes a callback for each element in the Set
values() Returns an iterator with all the values in a Set

Propertt Description
size Returns the number of elements in a Set

Differences between Set and Array

Feature Array Set
Purpose Ordered list of items Collection of unique items
Order Maintains order of elements Unordered
Access By index (e.g. myArray[0]) By Value (e.g. mySet.has())
Duplicates Allowed Not allowed

Maps

A Map holds key-value pairs where the keys can be any data type
A Map remembers the original insertion order of the keys.

Essential Map Methods

Method Description
new Map() Creates a new Map
set() Sets the value for a key in a Map
get() Gets the value for a key in a Map
entries() Returns an iterator with the [key, value] pairs in a Map

Differences between Objects and Maps

Object Map
Iterable Not directly iterable Directly iterable
Size Do not have a size property Have a size property
Key Types Keys must be Strings (or Symbols) Keys can be any datatype
Key Order Keys are not well ordered Keys are ordered by insertion
Defaults Have default keys Do not have default keys
