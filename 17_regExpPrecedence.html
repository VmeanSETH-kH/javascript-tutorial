Regular Expressions

A regular expression is a sequence of characters that forms a search pattern.
The search pattern can be used for text search and text replace operations.

Syntax


        /pattern/modifiers;

Example

    
        /nikolaTesla/i;

Example explained:

  • /nikolaTesla/i is a regular expression.
  • nikolaTesla is a pattern (to be used in a search).
  • i is a modifier (modifies the search to be case-sensitive)

Using String search() with a Regular Expression

Example

    
        let text = "Hi, Nikola";
        let n = text.search(/Nikola/i);     // n = 4

        let str = "Nikola Eistein !!"
        let result = text.replacec(/Nikola/i, "Albert"); 
        // result = "Albert Eistein !!"

Regular Expression modifiers

Modifiers can be used to perform case-sensitive more global searches:

Modifier Description
i Perform case-insensitive matching
g Perform a global match (find all)
m Perform multiline matching
d Perform start and end matching
